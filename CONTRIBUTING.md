# Contribuindo

Obrigado por considerar contribuir com este projeto.

## Requisitos

* Java 21+
* Maven 3.9+

## Executando o projeto

Clone o repositório:

git clone https://github.com/seu-usuário/Primeiro_Projeto_SpringBoot.git

Execute a aplicação:

./mvnw spring-boot:run

A API será iniciada em:

http://localhost:8080/api/v1/comidas

## Estilo de código

Este projeto segue as convenções Java

## Pull Requests

1. Faça um fork do repositório
2. Crie uma branch para a funcionalidade
3. Confirme suas alterações
4. Abra um Pull Request

## Problemas

Se você encontrar um bug ou quiser sugerir uma funcionalidade, abra um problema.
